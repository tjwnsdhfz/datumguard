# Contributor Covenant Code of Conduct

## 우리의 약속

우리는 연령, 신체 크기, 장애, 민족성, 성별 특성, 성 정체성과 표현, 경험 수준, 교육,
사회경제적 지위, 국적, 외모, 인종, 종교, 성적 정체성과 지향에 관계없이 모든 사람이 괴롭힘 없는
커뮤니티에 참여할 수 있도록 약속합니다.

우리는 개방적이고 환영하며 다양하고 포용적이고 건강한 커뮤니티에 기여하는 방식으로 행동할 것을
약속합니다.

## 기대하는 행동

긍정적인 환경에 기여하는 행동의 예는 다음과 같습니다.

- 친절과 공감을 보이기
- 서로 다른 관점, 경험, engineering discipline을 존중하기
- 건설적인 피드백을 주고받기
- 실수를 인정하고 영향받은 사람에게 사과하며 개선하기
- 개인의 승리보다 커뮤니티와 사용자의 안전에 최선인 것을 선택하기
- 주장보다 재현 가능한 contract, test, measurement evidence를 우선하기

허용되지 않는 행동의 예는 다음과 같습니다.

- 성적인 언어, 이미지, 원치 않는 관심 또는 접근
- 모욕, 비하, 선동적 발언과 인신공격
- 공개 또는 비공개 괴롭힘
- 명시적 허가 없이 주소, 이메일 등 타인의 개인정보를 공개하기
- 소유권이나 재배포 권한이 없는 비공개 CAD/BIM, 고객 정보, 시설 정보를 공유하기
- 전문적인 환경에서 부적절하다고 합리적으로 판단되는 그 밖의 행동

기술적 의견 차이는 사람에 대한 평가가 아닙니다. 구조 안전, 제작 적합성처럼 영향이 큰 주제에서는
근거와 한계를 구체적으로 밝히고 DatumGuard의 preliminary screening 결과를 certification처럼
과장하지 않습니다.

## 운영자의 책임

프로젝트 운영자는 허용되는 행동의 기준을 명확히 하고, 부적절하거나 위협적이거나 공격적이거나
유해하다고 판단되는 행동에 공정하고 적절한 조치를 취할 책임이 있습니다.

운영자는 이 Code of Conduct에 맞지 않는 comment, commit, code, wiki edit, issue 등의 기여를
삭제·수정·거부할 수 있으며, 필요한 경우 일시적 또는 영구적으로 참여를 제한할 수 있습니다.
조치의 이유는 적절한 범위에서 설명합니다.

## 적용 범위

이 Code of Conduct는 repository, issue, pull request, Discussion, review, 공개 demo와 프로젝트를
대표하는 온·오프라인 공간에 적용됩니다. 개인 계정이나 행사에서 프로젝트를 공식적으로 대표할
때에도 적용됩니다.

## 신고와 집행

가학적이거나 괴롭히거나 그 밖에 허용되지 않는 행동은 repository owner `tjwnsdhfz`에게 신고할
수 있습니다. 공개하기 어려운 내용은
[private security advisory](https://github.com/tjwnsdhfz/datumguard/security/advisories/new)를
열고 제목을 `[CODE OF CONDUCT]`로 시작해 주세요. 보안 취약점이 아니어도 기밀 신고 경로로
처리합니다. 즉각적인 신체적 위험이 있으면 먼저 해당 지역의 긴급 기관에 연락하세요.

모든 신고는 신속하고 공정하게 검토하며, 신고자의 개인정보와 안전을 존중합니다. 운영자는 집행
결정에 관여한 이해관계를 공개하고 필요한 경우 판단에서 물러납니다.

## 집행 지침

운영자는 위반의 심각성과 반복 여부를 고려해 다음 단계를 적용할 수 있습니다.

### 1. 시정

커뮤니티 기준에 맞지 않는 언어 또는 행동에 대한 비공개 설명과 수정 요청입니다. 공개 사과가
필요할 수 있습니다.

### 2. 경고

위반 결과와 반복 시 조치를 알리는 공식 경고입니다. 정해진 기간 동안 관련 당사자와의 상호작용을
제한할 수 있습니다.

### 3. 일시적 제한

정해진 기간 동안 프로젝트 공간과 커뮤니티 상호작용 참여를 제한합니다. 제한 기간에는 운영자에게
연락해 영향력을 행사하려는 시도도 허용되지 않습니다.

### 4. 영구적 제한

지속적인 부적절 행동, 괴롭힘 또는 개인에 대한 공격이 반복되면 프로젝트 커뮤니티 참여를
영구적으로 제한합니다.
